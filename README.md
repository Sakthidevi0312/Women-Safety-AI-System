# AI-Driven Women Safety Surveillance System

## Overview
This project is an AI-powered women safety monitoring system using:
- OpenCV
- MediaPipe
- MQTT
- Twilio
- MySQL

The system detects:
- Female alone at night
- Female surrounded by multiple males
- Distress hand gestures
- Danger zones

It sends:
- SMS alerts
- MQTT alerts
- Database logs

---

## Features
- Real-time face detection
- Age and gender detection
- Gesture recognition
- Emergency SMS alerts
- MQTT IoT communication
- MySQL logging
- Danger zone detection

---

## Technologies Used
- Python
